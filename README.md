# kin-starter-react-native

A starter project for React Native

## Installation

```sh
git clone https://github.com/kintegrate/kin-starter-react-native
cd kin-starter-react-native
yarn install
```

### iOS

```sh
cd ios && pod install
```

## Usage

Start the react-native 'metro' server:

```sh
yarn start
```

### Start Android app

```shell
yarn android
```

### Start iOS app

```shell
yarn ios
```
